Documentation

Zeend um framework fácil de ser usado e editado conforme sua necessidade. Cada página tem seu arquivo CSS específico, e cada exemplo logo abaixo tem as dicas de como deve ser feito e quais as classes que devem ser utilizadas em seus devidos lugares.

E como cada página tem seu arquivo CSS, não é necessária utilizar todos os arquivos CSS, sendo necessário apenas chamar os estilos utilizados nas páginas.

Seguem os exemplos de como devem ser importados os arquivos css em seu CSS padrão, fazendo essa chamada é só colocar o CSS utilizado dentro da pasta raiz css e pronto.
