Developed by designer crazy

Desenvolvido a partir da idéia de um web designer e que foi alimentada pela criatividade e necessidade de criar um framework fácil de ser utilizado e funções globais para serem usadas em qualquer parte do site.

CSS Editable

CSS fácil de ser editado e adaptado a necessidade do seu sistemas / site. Vários arquivos arquivos separados para serem utilizados apenas os que você quiser sem acumulo de CSS, o que aumenta o tempo de carregamento do site.

Cross Browser

Refere - se à habilidade de um site, aplicação web suportar múltiplos navegadores. Esta aplicação deve ser construída através de uma forma de programação que utiliza tecnologias compatíveis com qualquer navegador web que suporte as especificações do W3C.
